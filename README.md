# API to News System

Project made using ReactJS + Redux + TS + TailwindCSS + VITE.

# Get Starter

## Install Dependencies

In the root of the project, run the command:

```bash
npm install
```

## Run Project

Yet in the root of the project, run the command:

```bash
npm run dev
```
